# basic_web01
采用Spring5+SpringMVC5+Mybatis3+Shiro搭建的一套web基本架构，提供标准的RestFul API接口，可以很方便的进行二次开发！

>## 服务器端通用响应类封装

测试URL地址
http://localhost:8080/hello/abc

主要参数
```
SUCCESS(0, "成功"),
ERROR(1, "失败"),
ILLEGAL_ARGUMENT(2, "参数错误");
```
